### 2.5 后端关键设计

- **统一响应格式**：所有接口返回 `{code: 200, msg: "success", data: ...}`
- **全局跨域**：通过 CorsFilter 允许所有来源访问
- **逻辑删除**：使用 MyBatis-Plus `@TableLogic`，数据不物理删除
- **阅读量自增**：每次获取详情时 `view_count + 1`
- **双数据库支持**：开发用 H2（零配置），生产改配置项即可切换到 MySQL
- **Maven Wrapper**：项目自带 `mvnw`，不依赖系统全局 Maven 版本

### 3.4 核心组件详解

#### StarBackground（星空背景）
- 基于 `@tsparticles/react` + `@tsparticles/slim`
- 全屏固定定位 `z-index: -10`
- 120 颗粒子，颜色渐变（浅蓝/紫/白）
- 粒子间弱连接线，鼠标悬停抓取效果
- 移动端自动适配性能

#### Marquee（跑马灯）
- MagicUI 风格的自定义实现
- 支持水平/垂直滚动、反向滚动、悬停暂停
- 通过 CSS 变量 `--duration` 控制速度
- 左右渐变遮罩营造星空融合感

#### TechMarquee（技术栈卡片）
- 毛玻璃卡片（`backdrop-blur`）+ 发光边框
- hover 时 `shadow-glow` 悬浮发光
- 双层滚动（正向 + 反向），层次感强

#### ArticleCard（文章卡片）
- Framer Motion 入场动画（ staggered 延迟）
- 悬浮上浮 + 缩放动效
- 发光边框 + 毛玻璃背景
- 标签彩色胶囊、置顶标识、阅读量显示

#### Navbar（导航栏）
- 固定顶部，毛玻璃效果
- 路由激活项指示器动画（`layoutId`）
- 响应式移动端汉堡菜单（AnimatePresence）
- 集成主题切换按钮

#### Sidebar（侧边栏）
- 分类列表（含文章数徽标）
- 最新文章（前5篇，含日期）
- 标签云（彩色胶囊）
- 加载骨架屏状态

#### ThemeToggle（主题切换）
- 深色模式 / 浅色模式切换
- 状态持久化到 localStorage
- 弹簧动画过渡
